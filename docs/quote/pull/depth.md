获取标的的盘口

:::info

协议指令：`14`

:::

## Request

### Parameters

| 名称   | 类型   | 必须 | 描述                        | 示例       |
| ------ | ------ | ---- | --------------------------- | ---------- |
| symbol | string | 是   | 标的代码，`ticker.region`。 | `00700.HK` |

### Protobuf

```protobuf
message SecurityRequest {
  string symbol = 1;
}
```

## Response

### Response Properties

| 名称       | 类型     | 描述     |
| ---------- | -------- | -------- |
| symbol     | string   | 标的代码 |
| ask        | object[] | 卖盘     |
| ∟position  | int32    | 档位     |
| ∟price     | string   | 价格     |
| ∟volume    | int64    | 挂单辆   |
| ∟order_num | int64    | 订单数量 |
| bid        | object[] | 买盘     |
| ∟position  | int32    | 档位     |
| ∟price     | string   | 价格     |
| ∟volume    | int64    | 挂单辆   |
| ∟order_num | int64    | 订单数量 |

### Protobuf

```protobuf
message SecurityDepthResponse {
  string symbol = 1;
  repeated Depth ask = 2;
  repeated Depth bid = 3;
}

message Depth {
  int32 position = 1;
  string price = 2;
  int64 volume = 3;
  int64 order_num = 4;
}
```

## 接口限制

:::caution

- 每秒平均请求次数 10。
- 瞬时并发次数 5。

:::
